Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The meta is for info about the site and helps assisted devices.
  • The head section will include important info about the page.
  • The head is different from the header.

CSS

  • This page is to create style of the page.
  • This is attached to HTML as a style sheet.
  • CSS uses properties such as flex and color as well as heigth and width.
  • You can change the look of the page directly through CSS.
  • Class is used to describe the element and reference to CSS sheet.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • JavaScript is not the same as Java.
  • This page is also linked to HTML and can manipulate CSS and HTML.
  • JavaScript is what makes the page functional.
  • Commonly used with IF and FOR or LET statements.
